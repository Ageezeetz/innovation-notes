OpenAI DevDay 2025
	- Started off with introduction to ChatGPT's App API, where you can type a specific app and ask ChatGPT to do specific functions in the app (Spotify, create a playlist for chill music) ([Introducing apps in ChatGPT and the new Apps SDK | OpenAI](https://openai.com/index/introducing-apps-in-chatgpt/))
	- Next was a new release of AgentKit, which they used to showcase the creation of an AI agent within 8 minutes ([Introducing AgentKit | OpenAI](https://openai.com/index/introducing-agentkit/))
	- Then was Codex and announcing its beta release on Oct 6. Codex is what powers Github Copilot and is basically what turns your prompts into code, so they essentially completely revamped it. ([OpenAI Codex | OpenAI](https://openai.com/index/openai-codex/))
	- A demo was done using Codex to program a controller to pan a camera. While that was being created, they also created a voice assistant to control the lights, the assistant being capable of having conversations with the person and was shining lights at the audience.
	- Finally, ended things off with the API release of GPT mini and the API release of Sora
AMD signs chip deal with OpenAI
